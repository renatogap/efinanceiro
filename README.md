## Deploy Automatico (HostGator + GitHub Actions)

Este projeto ja possui workflow para deploy automatico em push na branch `main`.

Arquivo: `.github/workflows/deploy-hostgator.yml`

### 1) Configurar os Secrets no GitHub

No repositorio, acesse `Settings > Secrets and variables > Actions` e crie:

- `HOSTGATOR_HOST`: host SSH da HostGator (ex.: `111.222.333.444`)
- `HOSTGATOR_PORT`: porta SSH (normalmente `22`)
- `HOSTGATOR_USER`: usuario SSH
- `HOSTGATOR_SSH_KEY`: chave privada SSH do servidor
- `HOSTGATOR_DEPLOY_PATH`: pasta de deploy no servidor

Para seu dominio `efinanceiro.renatopereiradevx.com.br`, normalmente o caminho e algo como:

- `/home/SEU_USUARIO/efinanceiro`

E no cPanel/subdominio, o Document Root deve apontar para:

- `/home/SEU_USUARIO/efinanceiro/public`

### 2) Fluxo de deploy

Quando houver push na `main`, o workflow:

1. Faz build dos assets (`npm run build`)
2. Instala dependencias PHP
3. Envia arquivos via `rsync` por SSH
4. Executa comandos remotos do Laravel (`migrate`, `optimize`)

### 3) Observacoes importantes

- O arquivo `.env` do servidor **nao** e sobrescrito no deploy.
- Ajuste as variaveis de ambiente direto no servidor de hospedagem.
- Se quiser acionar manualmente, use `Actions > Deploy HostGator > Run workflow`.

## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.
